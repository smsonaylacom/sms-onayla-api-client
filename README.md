# Sms-onayla.com Python API Client

This is a lightweight Python wrapper for the **[Sms-onayla.com](https://sms-onayla.com)** API. It allows developers to integrate automated SMS verification services into their applications.

Easily receive SMS codes for services like **WhatsApp**, **Telegram**, **Instagram**, **Google**, **Facebook**, and more using virtual numbers.
